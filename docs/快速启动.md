# 快速启动

本文档面向老师验收、答辩演示和首次查看本项目的同学，提供最短路径的启动方式。

## 1. 进入项目目录

```bash
cd "/Users/pipi/Desktop/股票分析预测模型/李凯迪_2023111651_股票舆情风险预警与回测系统-提交版"
```

## 2. 生成默认演示工件

首次运行或发现 `data/processed/` 下缺少 `parquet` 文件时，先执行：

```bash
python3 scripts/build_demo_processed_assets.py
```

该命令会补齐默认演示所需的标准工件，让后端接口和前端页面都能直接读取数据。

## 3. 启动后端

```bash
python3 -m uvicorn backend.app:app --host 127.0.0.1 --port 8000
```

后端健康检查：

```bash
curl -s http://127.0.0.1:8000/api/meta/health
```

正常返回：

```json
{"status":"ok"}
```

## 4. 启动前端

新开一个终端执行：

```bash
cd "/Users/pipi/Desktop/股票分析预测模型/李凯迪_2023111651_股票舆情风险预警与回测系统-提交版/frontend"
npm install
npm run dev
```

浏览器访问：

```text
http://127.0.0.1:5173
```

## 5. 推荐答辩查看顺序

1. 首页指挥大屏  
   展示热度榜、焦点股、风险事件、AI 简报。

2. 新闻页  
   展示真实新闻流、来源覆盖情报、驱动词。

3. 个股详情页  
   展示成员 B 的情绪、主题和风险分析结果。

4. 回测页  
   展示原系统保留的历史验证能力。

```bash

```

```bash

```

### 成员 B 算法结果复查


```bash

```

输出文件：

## 9. 常见问题

### 1. `ModuleNotFoundError: No module named 'backend'`

原因：没有在项目根目录运行命令。  
解决：先执行第 1 步进入项目根目录。

### 2. 前端打不开

确认：

```bash
cd frontend
npm run dev
```

并访问：

```text
http://127.0.0.1:5173
```

### 3. 简报脚本找不到成员 B 的结果

当前成员 B 已覆盖股票代码：

- `600900`
- `000539`
- `000690`
- `000791`
- `000875`

若传入其他代码，脚本会提示“当前标的暂无成员 B 分析结果”。
