# 提交说明与启动指南

项目名称：面向上市公司舆情风控的多源中文大数据分析与研判系统

## 1. 文件夹说明

本提交包已经去除 `node_modules`、前端构建产物、缓存文件和本机临时文件。核心目录如下：

| 目录/文件 | 作用 |
| --- | --- |
| `frontend/` | React + Vite + Recharts 正式前端 |
| `backend/` | FastAPI 后端接口 |
| `src/` | 数据处理、情绪、预警、回测等核心模块 |
| `scripts/` | 数据处理和复现脚本 |
| `submission_final/` | 成员 A 数据包、数据字典、来源覆盖、DuckDB |
| `member_B/` | 成员 B 情绪、主题、风险、预测与图表结果 |
| `reports/` | 项目报告、PPT 口播、图表素材 |
| `docs/` | 成员计划、融合报告、快速启动和交付说明 |
| `llm_analyst_client.py` | AI 研判简报生成脚本 |
| `briefing_template.md` | AI 简报模板 |
| `requirements.txt` | Python 依赖 |

## 2. 环境要求

- Python 3.10 及以上
- Node.js 18 及以上
- npm

建议使用 macOS 或 Linux。Windows 可用，但路径命令需要自行调整。

## 3. 安装依赖

在项目根目录执行：

```bash
pip install -r requirements.txt
```

进入前端目录安装依赖：

```bash
cd frontend
npm install
```

安装后回到项目根目录：

```bash
cd ..
```

## 4. 启动后端

在项目根目录执行：

```bash
python3 -m uvicorn backend.app:app --host 127.0.0.1 --port 8000
```

后端健康检查：

```bash
curl http://127.0.0.1:8000/api/meta/health
```

如果返回：

```json
{"status":"ok"}
```

说明后端启动成功。

## 5. 可选：启用真实大模型 API

如果不配置 API Key，系统会使用本地模板生成 AI 研判简报，前端仍可正常展示。

如果要启用真实大模型调用，在启动后端前设置环境变量：

```bash
export LLM_API_KEY="你的大模型API_KEY"
export LLM_API_BASE="https://api.deepseek.com"
export LLM_MODEL="deepseek-chat"

python3 -m uvicorn backend.app:app --host 127.0.0.1 --port 8000
```

验证 AI 简报是否为真实 API 生成：

```bash
curl "http://127.0.0.1:8000/api/meta/llm-brief?ticker=600900.SH"
```

如果返回：

```json
"generation_mode": "remote_llm_a_b_fusion"
```

说明已经成功调用真实大模型。

如果返回：

```json
"generation_mode": "local_template_a_b_fusion"
```

说明系统使用本地模板兜底。常见原因是没有设置 `LLM_API_KEY`、Key 失效或网络不可用。

注意：不要把 API Key 写进代码、文档或提交包。

## 6. 启动前端

另开一个终端，进入前端目录：

```bash
cd frontend
VITE_API_BASE_URL=http://127.0.0.1:8000 npm run dev -- --host 127.0.0.1 --port 5173
```

浏览器访问：

```text
http://127.0.0.1:5173
```

如果 5173 端口被占用，改用 5174：

```bash
VITE_API_BASE_URL=http://127.0.0.1:8000 npm run dev -- --host 127.0.0.1 --port 5174
```

然后访问：

```text
http://127.0.0.1:5174
```

## 7. 推荐演示顺序

1. 首页指挥总览：展示热度榜、焦点股、风险事件、AI 研判简报。
2. 单股详情：展示个股画像、风险时间线、B 算法结果、增强预测。
3. 舆情钻取：展示新闻/公告/股吧样本、驱动词和来源覆盖。
4. 联动验证：展示价格、情绪、新闻量的联动关系。
5. 回测验证：展示历史信号验证和算法风险附证。

## 10. 常见问题

### 页面无法访问

先确认前端是否启动成功。终端应显示类似：

```text
Local: http://127.0.0.1:5173/
```

如果没有，重新启动前端。

### 页面数据显示为空

确认后端是否启动：

```bash
curl http://127.0.0.1:8000/api/meta/health
```

如果后端未启动，前端会无法读取数据。

### AI 简报还是模板

执行：

```bash
curl "http://127.0.0.1:8000/api/meta/llm-brief?ticker=600900.SH"
```

查看 `generation_mode`。如果是 `local_template_a_b_fusion`，说明没有成功调用远程大模型，但本地模板兜底正常。

### 端口被占用

后端可换端口，例如：

```bash
python3 -m uvicorn backend.app:app --host 127.0.0.1 --port 8001
```

前端启动时要同步改 API 地址：

```bash
VITE_API_BASE_URL=http://127.0.0.1:8001 npm run dev -- --host 127.0.0.1 --port 5173
```
